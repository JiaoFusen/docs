## 安装 Codex CLI

### NPM 安装

```shellscript
npm install -g @openai/code
```

### Homebrew 安装

```shellscript
brew install codex
```

## 更新 Codex CLI

### NPM 更新

```shellscript
npm install -g @openai/codex@latest
```

### Homebrew 更新

```shellscript
brew upgrade codex
```
